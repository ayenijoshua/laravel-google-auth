## About Project

Laravel Google Authentication task was accomplished by using state of the art implementations
I used the Laravel framework for smoth execution and then installed the laravel socialite package/plugin
to interface with Google Auth authentication system.

I then logged in to my Google account to create my credentials with are then used in the PHP application
for connection to Google Auth API

I created GooglrLogin controller function for handling user actions
I created database migrations for my database definitions
I created Events functions and classes to handle OTP expiration after 30 seconds.

I finally created a smmoth User interface for user interaction.
